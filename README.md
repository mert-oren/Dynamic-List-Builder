# Listify

Listify is a lightweight list management application built with HTML, CSS, and Vanilla JavaScript. It allows users to create custom lists, add items dynamically, and remove both lists and items through an interactive interface.

## Features

- Dynamic list creation
- Dynamic item creation
- List deletion
- Item deletion
- Clean and minimal interface
- Input validation
- Beginner-friendly project structure

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Run the Project

Simply open `index.html` in your browser.

## Purpose

This project was built to practice:

- JavaScript DOM manipulation
- Event listeners
- Dynamic UI rendering
- Basic component-like structuring with plain JavaScript
